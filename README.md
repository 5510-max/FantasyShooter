# Fantasy Shooter - 奇幻射击

一款俯视角动作射击 + RPG成长元素的独立游戏，类似《奇幻射击》《冒险王》风格。

## 🎮 游戏特色

- ✅ **爽快射击** - WASD移动，鼠标瞄准射击
- ✅ **冲刺闪避** - 空格键冲刺，可躲避攻击
- ✅ **RPG成长** - 击杀敌人获得经验，升级变强
- ✅ **无尽挑战** - 敌人会越来越强，看你能否生存
- ✅ **完整框架** - 代码结构清晰，易于扩展

## 🚀 快速开始

### 1. 安装Godot
- 下载 [Godot 4.x](https://godotengine.org/)
- 解压到任意位置

### 2. 打开项目
- 打开Godot
- 点击"导入"
- 选择 `project.godot` 文件

### 3. 运行游戏
- 按 **F5** 或点击播放按钮
- 使用WASD移动，鼠标瞄准，左键射击，空格冲刺

## 📁 项目结构
