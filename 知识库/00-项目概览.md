# Fantasy Shooter - 奇幻射击游戏开发知识库

## 项目简介
- **类型**: 俯视角动作射击 + RPG成长
- **风格**: 类似《奇幻射击》《冒险王》
- **引擎**: Godot 4.x
- **目标平台**: Steam (PC)

## 核心玩法循环
```
击杀敌人 → 获得经验 → 升级变强 → 面对更强敌人
     ↑_________________________________|
```

## 已完成功能
- ✅ 玩家移动（WASD）
- ✅ 鼠标瞄准射击
- ✅ 冲刺闪避（空格）
- ✅ 敌人AI（追逐+攻击）
- ✅ 经验值系统
- ✅ 升级系统
- ✅ UI界面（血条、经验条、等级）

## 文件结构
```
FantasyShooter/
├── project.godot          # 项目配置文件
├── scripts/               # 所有GDScript代码
│   ├── player.gd         # 玩家控制
│   ├── bullet.gd         # 子弹
│   ├── enemy.gd          # 敌人
│   ├── exp_orb.gd        # 经验球
│   ├── enemy_spawner.gd  # 敌人生成器
│   ├── game_manager.gd   # 游戏管理器
│   └── ui_manager.gd     # UI管理器
├── scenes/               # 场景文件(.tscn)
├── assets/               # 美术资源
│   ├── sprites/         # 精灵图
│   ├── sounds/          # 音效
│   └── music/           # 音乐
└── 知识库/              # 本知识库文档
```
