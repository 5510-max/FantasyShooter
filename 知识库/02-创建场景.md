# 创建游戏场景

## 1. 创建主场景

### 步骤
1. 在Godot中，点击 **场景 → 新建场景**
2. 添加 **Node2D** 作为根节点，命名为 `Main`
3. 保存为 `res://scenes/main.tscn`

### 场景结构
```
Main (Node2D)
├── GameManager (Node) - 附加 game_manager.gd
│   └── UIManager (CanvasLayer) - 附加 ui_manager.gd
│       ├── HUD (Control)
│       │   ├── HealthBar (ProgressBar)
│       │   ├── ExpBar (ProgressBar)
│       │   ├── LevelLabel (Label)
│       │   └── TimeLabel (Label)
│       ├── LevelUpMessage (Label)
│       ├── GameOverScreen (Control)
│       └── PauseMenu (Control)
├── Player (CharacterBody2D) - 附加 player.gd，分组：player
│   ├── Sprite2D
│   ├── CollisionShape2D (CircleShape2D)
│   ├── ShootTimer (Timer)
│   ├── DashTimer (Timer)
│   └── DashCooldownTimer (Timer)
├── EnemySpawner (Node2D) - 附加 enemy_spawner.gd
│   └── SpawnTimer (Timer)
└── Camera2D
```

## 2. 创建玩家场景

1. 新建场景，根节点选 **CharacterBody2D**，命名为 `Player`
2. 添加子节点：
   - **Sprite2D** - 显示玩家图片
   - **CollisionShape2D** - 碰撞体
     - 形状选 `CircleShape2D`
     - 半径设为 16
   - **Timer** × 3 - 分别命名为 `ShootTimer`、`DashTimer`、`DashCooldownTimer`

3. 选中Player节点，在检查器右侧点击 **节点 → 分组**
   - 添加分组：`player`

4. 附加脚本 `player.gd`

5. 保存为 `res://scenes/player.tscn`

## 3. 创建子弹场景

1. 新建场景，根节点选 **Area2D**，命名为 `Bullet`
2. 添加子节点：
   - **Sprite2D** - 子弹图片
   - **CollisionShape2D** - 碰撞体（RectangleShape2D）
3. 附加脚本 `bullet.gd`
4. 保存为 `res://scenes/bullet.tscn`

## 4. 创建敌人场景

1. 新建场景，根节点选 **CharacterBody2D**，命名为 `Enemy`
2. 添加子节点：
   - **Sprite2D** - 敌人图片
   - **CollisionShape2D** - 圆形碰撞体
3. 选中Enemy节点，添加分组：`enemies`
4. 附加脚本 `enemy.gd`
5. 保存为 `res://scenes/enemy.tscn`

## 5. 创建经验球场景

1. 新建场景，根节点选 **Area2D**，命名为 `ExpOrb`
2. 添加子节点：
   - **Sprite2D** - 经验球图片（可以用圆形）
   - **CollisionShape2D** - 圆形碰撞体
3. 附加脚本 `exp_orb.gd`
4. 保存为 `res://scenes/exp_orb.tscn`

## 6. 设置碰撞层

### 碰撞层说明
| 层 | 用途 |
|----|------|
| 1 | Player |
| 2 | Enemy |
| 3 | Bullet |
| 4 | Item |
| 5 | Wall |

### 设置方法
1. 选中节点
2. 在检查器中找到 **Collision → Layer** 和 **Mask**
3. Layer = 这个节点属于什么
4. Mask = 这个节点会检测什么

示例：
- **Player**: Layer=1, Mask=2,4,5（检测敌人、物品、墙）
- **Enemy**: Layer=2, Mask=1,3,5（检测玩家、子弹、墙）
- **Bullet**: Layer=3, Mask=2,5（检测敌人、墙）
- **ExpOrb**: Layer=4, Mask=1（只检测玩家）

## 7. 测试运行

1. 确保 `Main` 场景已保存
2. 按 **F5** 运行
3. 你应该能：
   - 用WASD移动
   - 鼠标瞄准
   - 左键射击
   - 空格冲刺
   - 击杀敌人获得经验
   - 升级后血量和伤害增加

## 常见问题

### Q: 玩家不移动？
A: 检查输入映射是否正确（项目 → 项目设置 → 输入映射）

### Q: 子弹不生成？
A: 确保 bullet.tscn 存在且路径正确

### Q: 敌人不追踪玩家？
A: 确保Player节点有 `player` 分组

### Q: 经验球不消失？
A: 检查ExpOrb的碰撞层和Mask设置
